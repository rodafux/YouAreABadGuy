# YouAreABadGuy

A BetterDiscord plugin designed to visually mark players you no longer want to team up with.

## Features

*   **Visual Indicators**: Marked users' nicknames appear in bright red across Discord (chat, member list, and voice channels).
*   **Local Records**: Save specific reasons for each marked user in your own private database.
*   **Notes Tooltip**: Hover over a marked name to display your saved reason and the date the user was added.
*   **Server-Based Management**: Export or import your "Bad Guys" list, with the ability to filter by specific servers.
*   **Multilingual Support**: Automatically detects your system language and supports English, French, German, Spanish, Italian, and Russian.

## How to Use

1.  **Marking a User**: Right-click a user and select "Mark as Bad Guy". A dialog will appear where you can enter an optional reason. Click "Save" to confirm.
2.  **Viewing Info**: Hover your mouse over a red-colored nickname to see the recorded reason and date added.
3.  **Managing the List**: Access the plugin settings to manually change the language or to export and import your lists for backup or sharing.
