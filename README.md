# YouAreABadGuy

A BetterDiscord plugin designed to help you keep track of toxic players or teammates you'd rather not play with again.

## Features

- **Visual Indicators**: Marked users' nicknames appear in glowing red throughout Discord (chat, member list, voice channels).
- **Local Records**: Save specific reasons for each marked user in your own private database.
- **Custom Tooltips**: Hover over a marked name to see your private notes in a polished, Discord-themed tooltip.
- **Server-Based Management**: Export or import your "Bad Guy" list filtered by specific servers you've visited.
- **Multi-language Support**: Automatically detects your system language with support for English, French, German, Spanish, Italian, and Russian.

## How to Use

- **Marking**: Right-click any user and select **Mark as Bad Guy**. A dialog will appear immediately—just type your reason and press Save.
- **Viewing Notes**: Hover your mouse over any red-colored nickname to reveal the recorded reason and the date the user was added to your list.
- **Management**: Access the plugin settings to switch languages manually or to export/import your lists for backup or sharing.
